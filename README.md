To run this project locally:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/KimRuedasR/rick-and-morty-browser
    ```


4.  **Run the development server:**
    ```bash
    npm run dev
    ```
    The application will be available at `http://localhost:5173`.
